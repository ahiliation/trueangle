== Frequently Asked Questions ==
* How do i add logo in the header?
Logo can be uploaded via Apperence >> Header page from your Dashboard.  The size of image used here is 300px width and 100px height. And the background color is #00aced.

* How do i add Favicon icon?
Favicon icon can be uploaded via ftp to your hosting and then get the url of favicon icon and then paste it into the Theme Options page. You can visit Theme Options page via Apperence >> Theme Options.

* How to show the post thumbnail in Homepage?
You can select Show Post Thumbnail through Theme Options page. You can change the size of Post Thumbnail through Media Page.

* How can i setup Soical Media Icon?
It can be done from Theme Options page just paste the profile url(including http://) of your respected Social Site.
